<Aside title="Browser Usage" type="tip">
TypeScript is not meant to run directly in the browser.

You might want to checkout [the JavaScript SDK](/sdks/javascript) for this case.
</Aside>

## Usage

```typescript
// Import the SDK
import TCGdex from '@tcgdex/sdk'

// Instantiate the SDK
const tcgdex = new TCGdex('en');

// go into an async context
(async () => {
	// Card will be Furret from the Darkness Ablaze Set
	const card = await tcgdex.fetch('cards', 'swsh3-136');
})();
```

You can check in [the JSON REST API docs](/rest) and select the correct tab to see the command specific for your SDK
