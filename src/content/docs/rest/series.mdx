## Request
<Tabs syncKey="sdk">
	<TabItem label="PHP">
		```php
		$series = $tcgdex->fetchSeries();
		```
		_[See how to setup the PHP SDK](/sdks/php)_

		_SDK currently does not [support filtering](/rest/filtering-sorting-pagination)_
	</TabItem>

	<TabItem label="Javascript">
		```javascript
		const series = await tcgdex.fetch('series');
		```
		_[See how to setup the Javascript SDK](/sdks/javascript)_

		_SDK currently does not [support filtering](/rest/filtering-sorting-pagination)_
	</TabItem>
	<TabItem label="Typescript">

		_SDK currently does not [support filtering](/rest/filtering-sorting-pagination)_

	<TabItem label="Java">

## Sample Response

The request return [an array of `SerieBrief` objects](/reference/serie-brief)

```json
[
	{
		"id": "base",
		"name": "Base",
		"logo": "https://assets.tcgdex.net/en/base/base1/logo"
	},
	// ...
	{
		"id": "xy",
		"name": "XY",
		"logo": "https://assets.tcgdex.net/en/xy/xyp/logo"
	}
]
```
