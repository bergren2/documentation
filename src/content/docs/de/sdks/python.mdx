## Installation

Führe den folgenden Befehl aus, um das SDK zu installieren:

```bash
pip install tcgdex-sdk
```

## Verwendung

```python
from tcgdexsdk import TCGdex

tcgdex = TCGdex("de") # Du kannst auch `Language.DE` verwenden TCGdex(Language.DE)
res = await tcgdex.card.get("swsh1-136")
```

Du kannst die [Dokumentation der REST JSON API](/rest) einsehen und den richtigen Tab auswählen, um den spezifischen Befehl für dein SDK zu sehen.
