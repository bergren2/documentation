## Cosa è disponibile

<CardGrid stagger>
	<Card title="6 lingue e altre in arrivo" icon="heart">
	Tutta l'API è disponibile in 6 lingue!
	- Inglese
	- Francese
	- Tedesco
	- Portoghese (Brasiliano)
	- Italiano
	- Spagnolo
	- e [altre in arrivo!](https://github.com/tcgdex/cards-database/pull/481)
	</Card>
	<Card title="un'API REST JSON" icon="add-document">
		Un'[API REST JSON completa](/it/rest) per ottenere informazioni su qualsiasi carta

		- Endpoint delle carte
		- Endpoint dei set
		- Endpoint delle serie
		- Endpoint dei campi delle carte

		È supportata anche la filtrazione e la paginazione per le operazioni di elencazione

		Sono disponibili diversi SDK per aiutarti a iniziare
	</Card>
	<Card title="un'API GraphQL" icon="setting">
	L'[API GraphQL](/it/graphql) ti consente di fare le tue richieste
		- `Card` e `Cards` per recuperare carta(e)
		- `Set` e `Sets` per recuperare set
		- `Serie` e `Series` per recuperare serie

		Le query supportano la filtrazione e la paginazione
	</Card>
</CardGrid>
