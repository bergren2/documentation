## Installation

<Tabs syncKey='pouet'>
	<TabItem label="NPM">
		```bash
		npm i @tcgdex/sdk
		```
	</TabItem>

	<TabItem label="Bun">
		```bash
		bun i @tcgdex/sdk
		```
	</TabItem>

	<TabItem label="Yarn">
		```bash
		yarn add @tcgdex/sdk
		```
	</TabItem>

	<TabItem label="PNPM">
		```bash
		pnpm add @tcgdex/sdk
		```
	</TabItem>

	<TabItem label="Browser">
		```html
		<script src="https://cdn.jsdelivr.net/npm/@tcgdex/sdk@2/dist/tcgdex.browser.js"></script>
		```
	</TabItem>
</Tabs>

## Nutzung

<Tabs syncKey='pouet'>
	<TabItem label="Node/Bun">
		```javascript
		// Importieren Sie das SDK mit moduleJS
		import TCGdex from '@tcgdex/sdk'
		// ODER mit commonJS
		const TCGdex = require('@tcgdex/sdk').default

		// SDK instanziieren
		const tcgdex = new TCGdex('en');

		// In einen asynchronen Kontext wechseln
		(async () => {
			// Die Karte wird Furret aus dem Set Flammende Finsternis sein
			const card = await tcgdex.fetch('cards', 'swsh3-136');
		})();
		```
	</TabItem>

	<TabItem label="Browser">
		```html
		<script>
			// SDK instanziieren
			const tcgdex = new TCGdex('en');

			// In einen asynchronen Kontext wechseln
			;(async () => {
				// Die Karte wird Furret aus dem Set Flammende Finsternis sein
				const card = await tcgdex.fetch('cards', 'swsh3-136');
			})();
		</script>
		```
	</TabItem>
</Tabs>

Sie können die [REST JSON API-Dokumentation](/rest) einsehen und den richtigen Tab auswählen, um den spezifischen Befehl für Ihr SDK zu sehen.
