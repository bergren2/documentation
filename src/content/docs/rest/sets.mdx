## Request
<Tabs syncKey="sdk">
	<TabItem label="PHP">
		```php
		// Fetch the cards !
		$sets = $tcgdex->fetchSets();
		```
		_[See how to setup the PHP SDK](/sdks/php)_

		_SDK currently does not [support filtering](/rest/filtering-sorting-pagination)_
	</TabItem>

	<TabItem label="Javascript">
		```javascript
		// Fetch the cards !
		const sets = await tcgdex.fetch('sets');
		```
		_[See how to setup the Javascript SDK](/sdks/javascript)_

		_SDK currently does not [support filtering](/rest/filtering-sorting-pagination)_
	</TabItem>

	<TabItem label="Typescript">
		```typescript
		// Fetch the cards !
		const sets = await tcgdex.fetch('sets');
		```
		_[See how to setup the Typescript SDK](/sdks/typescript)_

		_SDK currently does not [support filtering](/rest/filtering-sorting-pagination)_

	</TabItem>

	<TabItem label="Java">
		```java
		// Fetch the cards !
		ArrayList<SetResume> sets = tcgdex.getchSets();
		```
		_[See how to setup the Java SDK](/sdks/java)_

		_SDK currently does not [support filtering](/rest/filtering-sorting-pagination)_
	</TabItem>

	<TabItem label="Kotlin">
		```kotlin
		// returns you a Card Class with every informations filled!
		val sets = tcgdex.getchSets()
		```
		_[See how to setup the Kotlin SDK](/sdks/kotlin)_

		_SDK currently does not [support filtering](/rest/filtering-sorting-pagination)_
	</TabItem>

	<TabItem label="cURL">
		```
		https://api.tcgdex.net/v2/en/sets
		```
		_[See how to prepare your URLs](/rest)_

		_[Filtering documentation](/rest/filtering-sorting-pagination)_
	</TabItem>
</Tabs>
