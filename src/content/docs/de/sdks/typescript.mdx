## Installation

<Tabs>
	<TabItem label="NPM">
		```bash
		npm i @tcgdex/sdk
		```
	</TabItem>

	<TabItem label="Bun">
		```bash
		bun i @tcgdex/sdk
		```
	</TabItem>

	<TabItem label="Yarn">
		```bash
		yarn add @tcgdex/sdk
		```
	</TabItem>

	<TabItem label="PNPM">
		```bash
		pnpm add @tcgdex/sdk
		```
	</TabItem>
</Tabs>

<Aside title="Verwendung im Browser" type="tip">
TypeScript ist nicht dafür gedacht, direkt im Browser ausgeführt zu werden.

Sie möchten möglicherweise [das JavaScript SDK](/sdks/javascript) für diesen Fall konsultieren.
</Aside>

## Verwendung

```typescript
// SDK importieren
import TCGdex from '@tcgdex/sdk'

// SDK instanziieren
const tcgdex = new TCGdex('de');

// in einen asynchronen Kontext eintreten
(async () => {
	// Die Karte wird Furret aus dem Set Darkness Ablaze sein
	const card = await tcgdex.fetch('cards', 'swsh3-136');
})();
```

Sie können die [Dokumentation der JSON REST API](/rest) einsehen und den entsprechenden Tab auswählen, um den spezifischen Befehl für Ihr SDK zu sehen.
